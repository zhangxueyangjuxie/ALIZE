Detailed Description

Definition of speaker re-segmentation behavior.

Author
LIA
Version
2.0
Date
june 2011

The resegmentation step is normally applied in order to refine the segmentation outputs (issued from the segmentation step) and to remove irrelevant speakers.
A new HMM is generated from the segmentation output and an iterative speaker model training/Viterbi decoding loop is launched. In contrast to the segmentation stage, here the MAP adaptation (coupled with a generic speech/background model) replaces the EM/ML algorithm for speaker model estimation since the segmentation step provides an initial distribution of speech segments among the different speakers detected.
For the resegmentation process, all the boundaries (except speech/non-speech boundaries) and segment labels are re-examined.

Build a new cluster and associated segments for the decoding step from an existing segment Server given as input.

Remarks
In this function, :
  • Segments named "nonspeech" are removed
  • the set of the remaining segments is re-named 'ToDecode0' and grouped together in a same cluster for further decoding In this way, each decoding iteration being based on this cluster, labels and boundaries are re-examined at each of them

Check :

  • whether some global speaker durations (over the segmentation) are under a limit value fixed in the configuration file. In this case, speakers are removed from the segmentation
  • whether the stop criteria (for adaptation/decoding step) are reached
Returns
True if Comparison between the last two segmentations (previous and current) are identical otherwise false
Remarks
The comparison between both segmentations is defined by :
  1. the difference in the number of speakers
  2. the difference in values of the Viterbi paths according to an epsilon value given in the configuration file
Attention
The parameter decodingEpsilon is required for step 2.
The value of the parameter decodingEpsilon defines the absolute margin of acceptable difference in values of Viterbi paths between both segmentations.

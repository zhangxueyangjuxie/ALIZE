Selection of suitable segment for adding a new speaker.

Returns
True if suitable segment is found otherwise false
Remarks

The method of selection depends on the selectionMethod parameter which the following values can be associated with:

  • 'firstSegmentFound' => Select the first segment which duration is over the selectionLength parameter
  • 'firstLimitedLengthSegmentFound' => Select the first segment which duration is over than the selectionLength parameter, but truncated to reflect the limitedLength parameter
  • 'longerSegmentFound' => Select the longest segment which duration is over the selectionLength parameter
  • 'longerLimitedLengthSegmentFound' => Select the longest segment which duration is over than the selectionLength parameter, but truncated to reflect the limitedLength parameter

A segment is considered as suitable if :

  • the length of segment is superior or equal to the value of the parameter selectionLength
  • the pre-selected segment is not already used
